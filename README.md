# Student Marks Predictor

This project predicts student marks based on study hours using Machine Learning.

## Problem Statement
Students often want to estimate their expected marks based on the number of hours they study.

## Solution
A Linear Regression model is used to predict marks from study hours.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## Dataset
The dataset contains two columns:
- Study Hours
- Marks

## Machine Learning Model
Linear Regression

## Example Prediction
Input:
Study Hours = 7

Output:
Expected Marks ≈ 70
